# Fire (100 Men vs Gorilla)

A blockchain-integrated battle game where players choose to control either 100 men or a single gorilla in an epic showdown.

![Fire Game](https://placeholder.svg?height=300&width=600)

## Overview

Fire is a turn-based battle game built with Next.js and integrated with Ethereum wallets. Players connect their wallet, choose their side (100 men or a gorilla), and engage in strategic combat with unique abilities and attack animations.

## Features

- **Wallet Integration**: Connect with Ethereum wallets (MetaMask, etc.)
- **Character Selection**: Choose to play as either 100 men or a single gorilla
- **Turn-based Combat**: Strategic gameplay with different attack types
- **SVG Animations**: Visually engaging battle animations
- **Battle Log**: Real-time updates of battle events
- **Token System**: In-game currency for upgrades and trading
- **NFT Marketplace**: Collect and trade upgrades as NFTs

## Technologies

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **UI Components**: shadcn/ui
- **Blockchain**: Ethereum, ethers.js
- **Animations**: SVG, CSS animations
- **Styling**: Tailwind CSS

## Game Mechanics

### Battle System
- **Normal Attack**: Basic damage with simple animations
- **Special Attack**: Higher damage with elaborate animations (10-second cooldown)
- **Defend**: Protective stance that reduces incoming damage

### Characters
- **100 Men**: Strength in numbers with coordinated attacks
- **Gorilla**: Raw power with devastating single-target attacks
