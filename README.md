# Fire: City-Building Trading Game

A web-based dApp where players mint districts as Zora Coins, trade them, and build a virtual city on Base Sepolia.

## Features
- Connect wallet (MetaMask) to retrieve account address automatically.
- Mint districts with name, ticker, and image (Craiyon AI or manual upload).
- View city grid of owned districts (name, ticker, image, trading volume).
- Trade district coins (buy/sell, mock data for MVP).
- Profile page with wallet address, districts, and city value (sum of trading volumes).
- Minimalist, colorful UI (coral, teal, lime, violet) with responsive navbar, notifications, and animations.
